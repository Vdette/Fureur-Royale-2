# Fureur Royale - Dashboard

Ce tableau de bord permet de visualiser les données du clan Clash of Clans "Fureur Royale" avec les membres, les guerres et les statistiques associées.

- **Membres** : Statistiques détaillées, dons, contributions, classement
- **Guerres** : Historique des guerres avec résultats et pourcentages
- **Design** : Inspiré de ClashSpot avec un affichage responsive
